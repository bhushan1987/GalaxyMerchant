*Setup the application :
 It is a simple maven project, with following structure:
    - merchant (Root)
        - galaxy
            - source
Junit version 4.11 is the only dependency.
Please import the maven project in IntelliJ or Eclipse.
Execute- mvn clean install

*Running the application :
Run Main.java to run the application.
The application expects full file path of input file as a command line argument.
Sample input file(input.txt) is available in project root directory.

*Assumptions made :
- Each line is terminated by a newline character.
- Roman Numerals must be specified in UPPERCASE letters.
- The instructions are case sensitive.
- Questions end with a question mark (?).

*Design :
Please look for UMLs folder under project root. It contains Block Diagram and Class diagram.
